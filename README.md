# Scaffold for API express

## Install

Clone this repository and run:
```
 yarn // for install dependencies
```
And
```
 yarn dev:server // for start api
```

## Features:

- Jest
- CORS
- TypeORM
- JWT
- Bcrypt
- DotENV
- Express Async Errors
- Eslint + Prettier
- TS node dev
- Multer

## Scripts:

- build
- dev:server
- start
- typeorm
- lint
- test
